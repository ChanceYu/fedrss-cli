# fedrss-cli
使用命令行，搜索最新前端技术文章

## 在线版

https://front-end-rss.vercel.app

![](https://img.shields.io/npm/v/fedrss-cli.svg?color=blue&style=popout-square)

## 安装
```
npm install -g fedrss-cli
```

## 使用
```
$ fed
```

```
$ fedrss
```
